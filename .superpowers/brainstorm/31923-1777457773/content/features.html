Design — Section 3 of 3: Features & Screens

Every screen, what it does, and key implementation notes.

Auth & Onboarding

Sign-in
Splash with logo + "Continue with Google" button. Triggers google_sign_in + Firebase Auth. On success, router redirects: new user → /onboarding, returning published user → /app.
Onboarding
(4 steps)
Name+age → Photo (image_picker + Storage upload) → Bio (180 char limit) → Interests (pick 3–6 from pool of 24) → Preview → Publish. Each step writes to Firestore. Step bar shows progress. Back navigation works across all steps.

Main App (Bottom Nav — 4 tabs)

Discover
2-column grid of published profiles (excludes self + paused). Heart button on each card toggles like. Tap card → full profile view. Real-time Firestore stream.
Chats
Lists mutual matches with last message preview + unread badge. Horizontal avatar row at top. Tap → chat thread. Chat unlocked only when both users have liked each other. Badge on tab icon for unread count.
Liked You
List of users who have liked the current user. Badge count on tab. Tap any → profile view.
You
Your profile summary — avatar, name/age, bio, interests. Buttons to Preview profile and Edit bio. Settings gear top-right.

Fullscreen Screens

Profile View
Square hero photo, name + age, city, bio, interest chips. Fixed bottom bar with Back (ghost) + Like (primary) buttons. Like button toggles; match toast fires if mutual.
Chat Thread
Bubble UI — outgoing coral, incoming white. Match card at top. Avatar grouping for consecutive messages. Send on tap or Enter (keyboard). Messages stream from Firestore subcollection in real time. Marks messages read on open.
Settings
Pause profile toggle (hides from feed), Edit profile photo, Who can see me. Notifications section (UI only for now). Sign out — clears auth state, redirects to sign-in.
Edit Bio
Standalone bio editor (180 char limit). Save writes directly to Firestore. Toast confirms save.

Cross-cutting

🎨 Theme: Bold Playful — Fraunces display, Plus Jakarta Sans body, lavender bg (#f2eeff), coral accent (#ff5a6e), large radii.
🍞 Toast: Floating pill at bottom — "Liked Maya", "It's a match! 🎉", "Bio updated", "Profile published". Auto-dismisses after 2.2s.
🔒 Auth guard: go_router redirect checks auth state on every navigation. Unauthenticated → /signin. Authenticated + no profile → /onboarding. Published → /app.
📱 Platform: Android primary (as requested). iOS-compatible but google-services.json setup covers Android; GoogleService-Info.plist needed separately for iOS.
