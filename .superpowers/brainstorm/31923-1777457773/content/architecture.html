Each feature: screens/ · providers/ · repository/

State Management (Riverpod)

🔐 authProvider
Stream<User?> from Firebase Auth. Drives router guard — unauthenticated users land on sign-in.
👤 myProfileProvider
AsyncNotifier that reads/writes the current user's Firestore doc. Onboarding writes here; You tab reads here.
🗂 feedProvider
Streams published profiles from Firestore, excluding self. Merges with local like state.
❤️ likesProvider
Manages like/unlike writes to Firestore. Exposes mutual match detection for chat unlock.
💬 chatsProvider
Streams matches + message threads from Firestore. Only available for mutual matches.
🔔 toastProvider
Simple StateProvider for transient toast messages (liked, matched, bio saved).
